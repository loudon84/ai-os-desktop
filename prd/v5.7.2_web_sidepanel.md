# v5.7.2_Desktop Web 模块侧面板功能重构 PRD

版本号：`v5.7.2_web_operator_panels_refactor`
目标模块：`src/renderer/src/screens/WebOperator/**`
改造边界：仅重构 Renderer 侧 WebOperator 面板组织结构，不改 Main / Preload / Browser IPC / WebContentsView 控制链路。

---

## 1. 当前源码结论

当前 `WebOperatorScreen.tsx` 仍直接导入并管理 `BrowserStatePanel`、`ScreenshotPanel`、`BrowserActionLog`、`PageStructurePanel`、`CrmEventPanel` 等面板组件。`ScreenshotPanel` 仍在 import 中存在，但 JSX 中已没有真正渲染。

当前 Screen 内部还保留了 `screenshotRef`，并且 `focusedPanel === "screenshot"` 时仍会尝试滚动到截图面板，但该面板在右侧实际 DOM 中已经不存在。

当前右侧面板仍由 `WebOperatorScreen.tsx` 直接拼装，`BrowserStatePanel`、`CrmEventPanel`、`PageStructurePanel`、`BrowserActionLog` 都写在 Screen JSX 内部，没有独立的 panels 模块封装。

当前 CSS 中，右侧面板样式仍集中在 `web-operator.css`，包括 `.web-operator-layout__side-panels`、`.web-operator-layout__panel`、`.web-operator-layout__panel--grow`，没有独立面板 CSS 文件。 

当前 `WebOperatorSideRail.tsx` 仍包含 `screenshot: Camera` 图标映射，且侧边导航数据仍来自 `SECONDARY_NAV_BY_WORKSPACE["web-operator"]`。 

当前 workspace 二级导航仍声明 `"web-operator": ["browser-state", "crm-context", "page-structure", "screenshot", "action-log"]`，未包含 `HermesTaskPanel`。

`HermesTaskPanel.tsx` 已存在，并且内部已经接入 `usePendingActions()`，可显示待确认动作并支持 `confirmAction / rejectAction`。它应纳入新的 WebOperator panels 模块。

    size: "md",

  {
    id: "hermes-task",
    labelKey: "navigation.hermesTask",
    size: "md",
  },
  {
    id: "page-structure",
    labelKey: "navigation.pageStructure",
    size: "lg",
  },
  {
    id: "action-log",
    labelKey: "navigation.actionLog",
    size: "lg",
  },
];

export function isWebOperatorPanelId(value: string): value is WebOperatorPanelId {
  return WEB_OPERATOR_PANEL_ORDER.some((item) => item.id === value);

### 3.4 新增 panels/WebOperatorPanels.tsx

```tsx
import { useEffect, useMemo, useRef } from "react";
import { BrowserStatePanel } from "../BrowserStatePanel";
import { CrmEventPanel } from "../CrmEventPanel";
import { HermesTaskPanel } from "../HermesTaskPanel";
import { PageStructurePanel } from "../PageStructurePanel";
import { BrowserActionLog } from "../BrowserActionLog";
import { WebOperatorPanelCard } from "./WebOperatorPanelCard";
import {
  normalizeWebOperatorPanelId,
  WEB_OPERATOR_PANEL_ORDER,
  type WebOperatorPanelId,
} from "./web-operator-panels-contract";
import "./web-operator-panels.css";

interface WebOperatorPanelsProps {
  focusedPanel: string;
  sideCollapsed: boolean;
  externalRefreshTrigger: number;
  onRefreshSnapshot: () => void;
}

type PanelRefMap = Record<WebOperatorPanelId, React.RefObject<HTMLDivElement>>;

export function WebOperatorPanels({
  focusedPanel,
  sideCollapsed,
  externalRefreshTrigger,
  onRefreshSnapshot,
}: WebOperatorPanelsProps): React.JSX.Element {
  const activePanel = normalizeWebOperatorPanelId(focusedPanel);

  const panelRefs = useMemo<PanelRefMap>(
    () => ({
      "browser-state": useRef<HTMLDivElement>(null),
      "crm-context": useRef<HTMLDivElement>(null),
      "hermes-task": useRef<HTMLDivElement>(null),
      "page-structure": useRef<HTMLDivElement>(null),
      "action-log": useRef<HTMLDivElement>(null),
    }),
    [],
  );

  useEffect(() => {
    if (sideCollapsed) return;
    panelRefs[activePanel]?.current?.scrollIntoView({
      behavior: "smooth",
      block: "nearest",
    });
  }, [activePanel, panelRefs, sideCollapsed]);

  return (
    <div className="web-operator-panels">
      {WEB_OPERATOR_PANEL_ORDER.map((panel) => (
        <WebOperatorPanelCard
          key={panel.id}
          panel={panel}
          focused={activePanel === panel.id}
          panelRef={panelRefs[panel.id]}
        >
          {panel.id === "browser-state" ? (
            <BrowserStatePanel className="web-operator-panels__content" />
          ) : null}

          {panel.id === "crm-context" ? (
            <CrmEventPanel
              className="web-operator-panels__content"
              onRefreshSnapshot={onRefreshSnapshot}
            />
          ) : null}

          {panel.id === "hermes-task" ? (
            <HermesTaskPanel className="web-operator-panels__content" />
          ) : null}

          {panel.id === "page-structure" ? (
            <PageStructurePanel
              externalRefreshTrigger={externalRefreshTrigger}
              className="web-operator-panels__content"
            />
          ) : null}

          {panel.id === "action-log" ? (
            <BrowserActionLog className="web-operator-panels__content" />
          ) : null}
        </WebOperatorPanelCard>
      ))}
    </div>
  );
}
```

注意：上面代码里 `useMemo` 内部不应直接调用 `useRef`。Cursor 实现时应改成显式 ref：

```tsx
const stateRef = useRef<HTMLDivElement>(null);
const crmRef = useRef<HTMLDivElement>(null);
const hermesTaskRef = useRef<HTMLDivElement>(null);
const structureRef = useRef<HTMLDivElement>(null);
const logRef = useRef<HTMLDivElement>(null);

const panelRefs: PanelRefMap = {
  "browser-state": stateRef,
  "crm-context": crmRef,
  "hermes-task": hermesTaskRef,
  "page-structure": structureRef,
  "action-log": logRef,
};
```

Cursor 必须使用第二种写法，避免违反 React Hooks 规则。
