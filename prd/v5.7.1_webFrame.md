# copilot-desktop ver5.7.1：CRM Web 页面主动上报与 Electron 双向控制能力 PRD

## 1. 版本定位

版本号：`ver5.7.1_crm_desktop_bridge`

目标模块：

```text
src/renderer/src/screens/WebOperator/**
src/main/browser/**

```

本版本在现有 WebOperator 浏览器核心能力上增加 CRM 页面主动上报与 Electron 双向控制能力。

核心原则：

1. CRM 页面不做频繁自动上报。
2. 只有用户在 CRM 页面主动点击指定按钮时，CRM JSSDK 才允许提交当前页面上下文。

4. CRM JSSDK 单独产物交付给 CRM Web 系统安装，不由 Electron 打包注入。
5. Electron 侧通过 WebContentsView 专用 preload、IPC、session、webFrameMain、MessageChannelMain 的组合能力完成校验、转发、模块打开、页面回控。
6. 业务系统页面与 Electron Main Process 之间禁止直接暴露 Node/Electron 能力。

## 5. 不使用本地 WebSocket Bridge 的结论

本版本默认不增加本地 WebSocket Bridge。

原因：

1. CRM 页面运行在 Electron WebContentsView 内部，已有 preload + IPC 的可信通道。
2. 上报不是高频流式数据，不需要 socket 长连接。
3. WebSocket Bridge 会引入端口占用、防火墙、CORS、鉴权、局域网暴露等额外安全面。
4. 现有项目已经有 IPC、ShellView、BrowserController、BrowserToolServer，不应再增加一套平行通信栈。

保留扩展条件：

只有在以下场景才允许后续增加 WebSocket Bridge：

1. CRM 必须在系统默认浏览器中打开，而不是 Electron WebOperator。
2. 多个外部浏览器标签页需要同时连接桌面端。
3. 需要本地桌面端作为系统级 URL Protocol / Socket 服务。
4. CRM 无法被 WebContentsView 承载。

## 6. 关键设计

### 6.1 WebOperator 专用 CRM preload

新增：

```text
src/preload/crm-bridge-preload.ts
```

职责：

1. 仅用于 WebOperator 外部 CRM WebContentsView。
2. 不复用主窗口 `src/preload/index.ts`。
3. 不暴露 `ipcRenderer`。
4. 不暴露 Node API。
5. 只暴露最小 CRM bridge API。
6. 监听 CRM JSSDK 的 `window.postMessage`。
7. 记录最近一次真实用户点击时间。
8. 对 CRM 上报事件做第一层校验。

preload 需要捕获真实用户手势：

```ts
let lastTrustedClickAt = 0;

window.addEventListener(
  "click",
  (event) => {
    if (event.isTrusted) {
      lastTrustedClickAt = Date.now();
    }
  },
  true,
);
```

只允许在用户点击后的时间窗口内提交：

```ts
const USER_GESTURE_WINDOW_MS = 1500;

function hasRecentTrustedGesture(): boolean {
  return Date.now() - lastTrustedClickAt <= USER_GESTURE_WINDOW_MS;
}
```

### 6.2 CRM JSSDK 与 preload 的通信方式

CRM JSSDK 优先调用：

```ts
window.CopilotDesktopCRM?.emit(event)
```

如果 Electron preload 未暴露 API，则退化为：

```ts
window.postMessage(
  {
    source: "copilot-crm-jssdk",
    channel: "crm.desktop.bridge",
    version: "0.1.0",
    event,
  },
  window.location.origin,
);
```

preload 同时支持两种入口：

1. `contextBridge.exposeInMainWorld("CopilotDesktopCRM", api)`

```ts
    ...input,
    app: "crm",
    url: input.url ?? window.location.href,
    title: input.title ?? document.title,
  };
}

async function emit(
  type: CrmDesktopEventType,
  page: CrmDesktopPageContext,
  payload?: Record<string, unknown>,
  options?: CrmDesktopEmitOptions,
): Promise<{ ok: boolean; message?: string }> {
  const event = {
    source: "crm-web",
    sdkVersion: SDK_VERSION,
    requestId: options?.requestId ?? createRequestId(),
    type,
    trigger: {
      type: "user-click",
      elementId: options?.triggerElementId,
      label: options?.triggerLabel,
      timestamp: new Date().toISOString(),
    },
    page: normalizePageContext(page),
    payload,
  };

  if (window.CopilotDesktopCRM?.emit) {
    return await window.CopilotDesktopCRM.emit(event);
  }

  window.postMessage(
    {
      source: "copilot-crm-jssdk",
      channel: MESSAGE_CHANNEL,
      version: SDK_VERSION,
      event,
    },
    window.location.origin,
  );

  return {
    ok: false,
    message: "Desktop bridge is not available in this browser context",
  };
}

function submitPageContext(
  page: CrmDesktopPageContext,
  payload?: Record<string, unknown>,
  options?: CrmDesktopEmitOptions,
): Promise<{ ok: boolean; message?: string }> {
  return emit("crm.context.submit", page, payload, options);
}

function onCommand(handler: (command: CrmDesktopCommand) => void): () => void {
  const listener = (event: MessageEvent) => {
    if (event.origin !== window.location.origin) return;
    const data = event.data;
    if (!data || data.source !== "copilot-desktop") return;
    if (data.channel !== COMMAND_CHANNEL) return;
    if (!data.command) return;
    handler(data.command as CrmDesktopCommand);
  };

  window.addEventListener("message", listener);
  return () => window.removeEventListener("message", listener);
}

function isDesktopAvailable(): boolean {
  return Boolean(window.CopilotDesktopCRM?.emit);
}

export const CopilotCrmDesktopSDK = {
  version: SDK_VERSION,
  emit,
  submitPageContext,
  onCommand,
  isDesktopAvailable,
};

window.CopilotCrmDesktopSDK = CopilotCrmDesktopSDK;
```

## 12. Electron preload 参考骨架

文件：

```text
src/preload/crm-bridge-preload.ts
```

```ts
import { contextBridge, ipcRenderer } from "electron";

const SDK_SOURCE = "copilot-crm-jssdk";
const EVENT_CHANNEL = "crm.desktop.bridge";
const COMMAND_CHANNEL = "crm.desktop.command";
const USER_GESTURE_WINDOW_MS = 1500;

let lastTrustedClickAt = 0;

window.addEventListener(
  "click",
  (event) => {
    if (event.isTrusted) {
      lastTrustedClickAt = Date.now();
    }
  },
  true,
);

function hasRecentTrustedGesture(): boolean {
  return Date.now() - lastTrustedClickAt <= USER_GESTURE_WINDOW_MS;
}

function assertTrustedGesture(): void {
  if (!hasRecentTrustedGesture()) {
    throw new Error("USER_GESTURE_REQUIRED");
  }
}

async function emit(event: unknown): Promise<{ ok: boolean; message?: string }> {
  try {
    assertTrustedGesture();
    return await ipcRenderer.invoke("crm-bridge:emit", {
      event,
      location: {
        href: window.location.href,
        origin: window.location.origin,
        title: document.title,
      },
      receivedAt: new Date().toISOString(),
    });
  } catch (error) {
    return {
      ok: false,
      message: error instanceof Error ? error.message : String(error),
    };
  }
}

contextBridge.exposeInMainWorld("CopilotDesktopCRM", {
  version: "0.1.0",
  isAvailable: () => true,
  emit,
});

window.addEventListener("message", (messageEvent) => {
  if (messageEvent.origin !== window.location.origin) return;

  const data = messageEvent.data;
  if (!data || data.source !== SDK_SOURCE) return;
  if (data.channel !== EVENT_CHANNEL) return;
  if (!data.event) return;

  void emit(data.event);
});

ipcRenderer.on("crm-bridge:command", (_event, command) => {
  window.postMessage(
    {
      source: "copilot-desktop",
      channel: COMMAND_CHANNEL,
      version: "0.1.0",
      command,
    },
    window.location.origin,
  );
});
```

## 13. Main Process IPC 参考骨架

文件：

```text
src/main/crm-bridge/crm-bridge-ipc.ts
```

```ts
import { ipcMain, WebContents } from "electron";
import { validateCrmBridgeEvent } from "./crm-bridge-security";
import { routeCrmBridgeEvent } from "./crm-event-router";
import { insertCrmBridgeEvent, listCrmBridgeEvents } from "./crm-event-store";
import { dispatchCrmCommand } from "./crm-command-dispatcher";

export function registerCrmBridgeIpc(): void {
  ipcMain.handle("crm-bridge:emit", async (event, input) => {
    const sender = event.sender;

    const validation = validateCrmBridgeEvent({
      sender,
      frame: event.senderFrame,
      input,
    });

    if (!validation.ok) {
      return validation;
    }

    const bridgeEvent = validation.event;

    insertCrmBridgeEvent({
      ...bridgeEvent,
      webContentsId: sender.id,
      createdAt: new Date().toISOString(),
    });

    const routeResult = await routeCrmBridgeEvent({
      sender,
      event: bridgeEvent,
    });

    return {
      ok: routeResult.ok,
      requestId: bridgeEvent.requestId,
      action: routeResult.action,
      message: routeResult.message,
    };
  });

  ipcMain.handle("crm-bridge:list-events", async (_event, limit = 20) => {
    return listCrmBridgeEvents(limit);
  });

  ipcMain.handle("crm-bridge:send-command", async (_event, command) => {
    return dispatchCrmCommand(command);
  });
}
```

## 15. 安全边界

### 15.1 禁止项

1. 禁止 CRM 页面直接访问 `ipcRenderer`。
2. 禁止 CRM 页面直接访问 Node API。
3. 禁止 Electron 给任意网站暴露 CRM bridge。
4. 禁止自动定时上报页面全文。
5. 禁止默认开启 WebSocket Bridge。
6. 禁止将 password、token、cookie、Authorization header 放入 CRM payload。
7. 禁止跨源 iframe 写操作。
8. 禁止未确认的敏感 click/type 操作。
9. 禁止 payload 超过配置大小。
10. 禁止 `eval` 执行 CRM 传入脚本。

1. origin allowlist。
2. event type allowlist。
3. user gesture 校验。
4. schema 校验。
5. payload size 校验。
6. requestId 去重。
7. 操作审计。
8. 错误码标准化。
9. WebOperator layer 校验。
10. preload 最小暴露。
